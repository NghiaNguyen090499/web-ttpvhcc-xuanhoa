## 3. DECISIONS LOG

| Quyết định | Lý do | Người chốt | Không đổi? |
|------------|-------|-------------|------------|
| Gemini 2.0 Flash | Nhanh, rẻ, tiếng Việt tốt | User + Agent | Có |
| Streaming SSE (không WebSocket) | Đơn giản, đủ cho chat 1-chiều | Agent | Có |
| Rate limit in-memory | Single instance Vercel, đủ chống spam | Agent | Không (cần Redis nếu scale) |
| Breakpoint `nav: 1100px` | `lg` (1024) quá nhỏ, `xl` (1280) quá lớn cho 7 mục | Agent | Có |
| Chatbot chỉ tham khảo | Yêu cầu user — không thay thế tư vấn chính thức | User | Có |
| Kiosk links mở tab mới | Không embed iframe — bảo mật + UX tốt hơn | Agent | Có |

## 7. CONTEXT ESSENTIALS

### Constraints
- **TailwindCSS v4** — dùng `@theme` thay `tailwind.config`
- **Next.js 16** — `await params` cho dynamic routes, không `ssr: false` trong Server Components
- **Turbopack** — không dùng `require()` cho JSON, dùng `fs.readFileSync`
- **Breakpoint nav**: `--breakpoint-nav: 1100px` — custom cho header

### Quy ước
- Data files: `data/*.json` (procedures, news, useful-links)
- Assets local: `public/images/`, `public/pdf/`
- Emoji thay FontAwesome — không phụ thuộc CDN
- Server Components mặc định — `"use client"` chỉ khi cần

### Rủi ro
- Chatbot trả lời sai thủ tục → disclaimer + fallback liên hệ nhân viên
- PDF dung lượng lớn (~65MB) → cần lazy-load tương lai
