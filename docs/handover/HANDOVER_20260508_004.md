## 2. ACHIEVEMENTS

### Deliverables

1. **25 ảnh preview** — copy local từ Canva designs, phục vụ hiển thị thumbnail
2. **25 file PDF** — tài liệu thủ tục đầy đủ, tải trực tiếp từ server
3. **Cập nhật procedures.json** — thêm trường `documents`, `preview`, `pdf`, `pageCount` cho mọi loại (cấp phường, phi địa giới, thường gặp)
4. **UI trang thủ tục** — hiển thị ảnh bìa + nút "Tải PDF (X trang)" cho từng lĩnh vực
5. **UI trang chi tiết** — section "Tài liệu thủ tục" với badge loại + preview + download
6. **Hoàn toàn độc lập Canva** — mọi asset local, không link ngoài
