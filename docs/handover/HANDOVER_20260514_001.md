## 7. CONTEXT ESSENTIALS

### Constraints
- Dữ liệu CRUD: JSON (`/data/`) — cần migrate khi có Supabase
- API routes ghi file JSON trực tiếp — không dùng cho production
- Middleware sắp deprecated trong Next.js 16

### Quy ước
- Server Components mặc định — `"use client"` chỉ khi cần (form, state)
- Admin pages dùng `"use client"` vì có state management
- API routes validate input trước khi xử lý
- Ghi chú mã nguồn tiếng Việt (@nhom, @chucnang, @input, @output)

### Rủi ro
- JSON write race condition khi nhiều admin cùng thao tác
- Connection limit Supabase free tier (max 60 connections)
