### Deliverables

## 7. CONTEXT ESSENTIALS

### Constraints
- **Next.js 16.2.6** (App Router, Turbopack)
- **Prisma v7.8.0** — bỏ `url` trong datasource, dùng `@prisma/adapter-pg`
- **jose** — JWT library, edge-compatible
- Middleware deprecated → proxy (warning only)

### Quy ước
- Auth: session cookie `admin-session`, TTL 8h
- Dev login: `admin@xuanhoa.gov.vn` / `admin123`
- Admin API: `/api/admin/*` — protected by middleware
- Admin CRUD: JSON file-based (dev), Prisma (production)

### Rủi ro
- Middleware deprecation có thể break trong Next.js tương lai
- JSON file CRUD không persist trên Vercel serverless
